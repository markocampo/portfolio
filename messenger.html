Facilitating communication with corporate teams

My Roles

  • UX Research
  • UX & UI Design

My Process

Define

Corporate teams use Dissolve's web tools to collaborate directly on the site. However, critical discussion is missed because they cannot instantly communicate on the platform.

  • What: Design an instant messenger feature
  • Who: Corporate teams who work directly on Dissolveâ€™s web app
  • Why: To facilitate communication between team members

Research

Reverse engineering existing applications lets me gain insights into UX & UI that has been rigorously tested. I studied Facebook and Instagram Messenger to analyze my own experience as a daily social media user. Then I observed Slack to understand why businesses(including my employer) prefer it as their main form of work communication. This method lets me adapt and customize features into my own design.

Components I discovered from my research that I adapted into my solution:

  • Sidebar - Contains primary user controls
  • Message box - Contains secondary user controls
  • Searchbar - Queries message recipients and existing chat sessions
  • Direct/private messaging
  • Group messaging
  • Alert system
  • Workflow integration
×

Design

To visualize the entire system, I outlined user touchpoints together with back-end system processes in a diagram.

Prototyping

Initial plans were to implement Messenger inside the web tool, but stakeholders agreed it should be available globally accross the site for easy access.

I proposed to insert Messenger in the main site navigation as a dropdown panel.

Sidebar Components

  • Searchbar
  • Direct messages section
  • Online/offline status
  • Group messages section
  • Default corporate chat - automaticlly created
  • New message alerts

Message box components

  • Header - displays the message recipient or group name
  • Messages section - reading messages
  • Textfield - entering messages
  • Send button - sends textfield input to recipient

Iterations

  • Added user avatars to aid user recognition
  • Added time stamps to messages
  • Added a dropdown menu to the groupname - contains tertiary user controls
  • Added group member count and avatars in the group message box header
  • Added a component to add new members in the group message box header

Create/edit group components

  • Group name field
  • Add members funtionality
  • Create/attach a workspace functionality - workflow integration

Validate

Designs were reviewed by the product manager and the customer success team who work directly with corporate accounts. Our in-house data analytics team will be capturing various data points on user touchpoints, and user feedback will be documented when the feature goes live. All captured data will enable us to define UX and UI issues that we'll address to improve the application.

Outcome

A lite version of messenger for corporate accounts was launched in May 2020. A total of 119 users has sent over 200 messages and created over 100 chat rooms. An overhauled version integrating Dissolve Creators is currently in development.
