Enabling creatives to share digital media on Dissolve's new platform

My Roles

  • UX Research
  • UX & UI Design
  • Project Management

My Process

Define

Dissolve is an ecommerce platform that sells stock footage and photography. Dissolve Creators is a project that aims to empower creatives around the globe by facilitating free media sharing online.

I kicked-off the project by meeting with stakeholders to gather insights about the business goals.

  • What: Design a functionality for uploading free media assets to Dissolve’s platform
  • Who: This functionality services creatives who share free media assets online and the people who utilize them in their personal or commercial projects
  • Why: To drive more traffic to the site by servicing a new niche audience, and to grow a free media library that can be monetized through a subscription business model and API connections.

Research

I conducted competitor research to empathise with the target audience and to identify opportunities for improvement.

Pain Points

I observed a few user journeys and identified pain points creatives encountered on existing platforms.

  1. Some creatives use existing platforms for portfolio purposes only, but there is no ability to disable public downloads
  2. No monetization tools – creatives resort to using third party applications to monetize their digital assets

Solutions

I reported my observations to the stakeholders and discussed potential solutions to user pain points.

  1. Develop a functionality that allows users to disable public downloads for their digital assets
  2. Facilitate C2C business by redirecting prospective sellers to Dissolve's monetization solutions

After confirming these additional requirements, I proceeded with the design.

×

Design

I went through the media upload process on competitor applications to gain insights into their information architecture. Then, I adapted my research to generate a custom user flow for Dissolve's media upload tool.

Prototyping

Designing in medium-high fidelity enabled stakeholders to provide more meaningful feedback and buy-in much easier. I utilized existing UI components in Dissolve's UI pattern library to efficiently design the mockups.

Free Tab

Key Components

  • Free tab
  • Upload button - Primary(blue) button color
  • Media thumbnails

Internal feedback

  • Draw more attention to the upload button
  • Upload button is in an awkward position
  • Insert marketing copy to inform users about this new functionality
  • The 'Free' tab copy isn't ideal for the brand image – Premium stock footage and photography

Iterations

  • Added a jumbotron component to highlight the upload functionality
  • Centered the upload button in the jumbotron component
  • Placed marketing copy in the jumbotron
  • Changed 'Free' tab copy to 'Uploads' and changed the active tab color to blue
  • Increased content section width

Upload Modal

Key components

  • Browse button for file selection
  • Drag and drop target area for dropping files
  • Link to the Upload Terms

Additional requirements requested

  • Add link to FAQ
  • Display the user's storage plan and upload restrictions to set up the subscription business model
  • Fascilitate user access to storage plans about page and upgrade page
  • Update 'Upload Terms' link copy

Iterations

  • Added FAQ icon beside the modal header – consistent with FAQ icon in the main navigation
  • Added user's storage plan and links to upgrade
  • Added upload restrictions info box
  • Changed 'Upload Terms' copy to 'Dissolve Free Content EULA'

Edit Modal

Key components

  • Media thumbnail
  • arrow icons – paginate between uploaded files
  • Metadata fields
  • Keywords
  • Downloadable checkbox – enable/disable downloads
  • Finish button – closes modal

User feedback

  • Users overlooking the 'Arrow' icons – Users attempt to paginate using the 'Finish' button
  • Users can't easily recognize their files

Marketing requests

  • Mock up placeholder text inside input fields
  • Replace checkbox copy with call to action – 'Allow public downloads'
  • Add tooltip for allowing public downloads

Iterations

  • Removed arrow icons – replaced with 'Next' and 'Back' buttons for pagination
  • 'Next' button transforms into 'Finish' button on the last item view
  • Added original filename under the thumbnail
  • Replaced checkbox copy
  • Added tooltip beside checkbox copy

Products Tab

The products tab integrates Dissolve's existing users and is an entry point for prospective sellers.

Key Components

  • Products tab
  • Commerce marketing elements in the jumbotron

Validate

Designs were tested internally with the content team who work directly with the target users. Dissolve's content team also stems from creative backgrounds, so their feedback was critical in the design reviews. The upload functionality is live and our team monitors live user data and feedback that are turned into action items to optimize the upload functionality.

Outcome

I gained long term experience working in an agile product environment which enabled me to make continuous improvements to the user experience throughout the feature's development cycle. Working in multiple cross-functional teams also allowed me to develop my team collaboration and project management skills.

Data Analytics

Dissolve Creators soft launched in January. So far, creatives have uploaded over 1000 free media assets to the platform without any marketing campaigns launched.
